# ChromeMomentum

## Info
Chrome의 Momentum 앱을 직접 구현한 JS 프로젝트입니다.

## Feature
1. Session을 이용한 Login
2. Login 상태 확인 검사 -> login UI를 띄울 것인지 Main UI를 띄울 것인지 정함
3. Main Ui : Todo, date, time, Random quotes, weather, Random mix color background
